## 📌 Problem Statement

People don’t fail because they forget deadlines — they fail because they don’t know how to execute them properly.

---

## 💡 Solution

Deadline Guardian AI turns every deadline into an execution plan:

- AI risk score (0–100)
- Smart daily breakdown
- Personalized reminder emails
- Automated n8n workflows
- Proactive deadline management

---

## ⚙️ Tech Stack

- Next.js (Frontend)
- Firebase Firestore (Database)
- Google Gemini AI (LLM)
- n8n (Automation)
- Gmail API (Emails)
- Google Cloud / Vercel (Deployment)

---

## 🧠 Architecture

User Input → Gemini AI → Firestore → n8n Trigger → Gemini Email → Gmail → Update Firestore

---

## 🚀 Features

- AI-based risk scoring
- Task planning automation
- Smart email reminders
- No duplicate alerts
- Dashboard analytics
